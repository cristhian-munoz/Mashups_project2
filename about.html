What does the page do?

  1. It is basically an info page for my mother, that displays weather forecasts that might be useful for her.
  2. The forecasts come along with recomendations, and the fact that the page displays 5 days of forecasts helps her to plan ahead.
  3. The page allows the user to change the location and update the forecasts accordingly.

API's used

  1. Accuweahter Index Forecast API: displays the forecasts for 5 consecutive days of a certain index. The indices used were: UV index, Arthritis Pain, and Lawn Mowing
  2. Accuweather Text Search API: this API provides the page with the 'location key' unique of the city the user is requesting forecasts from.
